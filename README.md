# 🌐 Statistically Intelligent — Portfolio Website

This repository contains the source code for my personal portfolio website built using **Quarto** and deployed via **GitHub Pages** with automated CI/CD using **GitHub Actions**.

The site showcases my work in **statistics, data analytics, machine learning, and intelligent automation**, along with blogs, projects, and technical explorations.

---

## ⚙️ How It Works

The deployment process is fully automated:

1. Changes are made in `.qmd` files using VS Code  
2. Changes are pushed to the `main` branch  
3. GitHub Actions automatically:
   - Renders the Quarto site  
   - Builds static HTML output  
   - Deploys to the `gh-pages` branch  
4. GitHub Pages serves the latest version of the site
